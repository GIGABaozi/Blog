## Variables

Local variables can be added into the HTML using the curly braces syntax:

```astro
---
const name = 'Astro'
---

<div>
	<h1>Hello {name}!</h1>
	<!-- Outputs <h1>Hello Astro!</h1> -->
</div>
```
